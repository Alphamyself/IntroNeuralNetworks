Getting Started

Download this git repository folder.
  - click the green button "Clone or Download"
  - click "Download Zip"
  - unzip folder

Download Anaconda here: https://www.anaconda.com/download/
Open anaconda and click Jupyter Notebook
In Jupyter Notebook file manager system, navigate to the folder you just downloaded
